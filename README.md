# Notas-atletas
## Projeto de certificação 1 - Pontuação dos atletas


## Descrição
O objetivo desse projeto é calcular a média das notas dos atletas, considerando apenas as 3 notas centrais do atleta.
As notas são organizadas de forma crescente e é extraido apenas as 3 notas do centro(1, X, X, X, 10)


## Estrutura de dados de entrada
O projeto trabalha com uma estrutura de dados composta por um array de objetos. Cada objeto contém o nome do atleta e suas respectivas notas


## Funcionamento
A função "dadosFinais" recebe um array de objetos, onde cada objeto representa um atleta e contém as suas notas. O objetivo da função é:

1° - Organizar as notas de cada atleta em ordem crescente.  
2° - Descartar a menor e a maior nota.  
3° - Calcular a média das 3 notas restantes.  
4° - Retornar um array contendo o nome do atleta, as notas ordenadas e a média calculada.
